python线程之队列Queue

Aug 8th, 2014 | Comments

Python的Queue模块中提供了同步的、线程安全的队列类.

1.FIFO（先入先出)队列Queue，
2.LIFO（后入先出）队列LifoQueue，
3.优先级队列PriorityQueue。

这些队列都实现了锁原语，能够在多线程中直接使用。可以使用队列来实现线程间的同步。
