Objective-C-Clang预编译宏

August 16, 2014
xcode

针对Objective-C-,Clang编译器内置哪些宏呢？

像平时看到的

__BLOCK__:是否支持block?

__unsafe_unretained:LLVM 编译器 3.0版本之后,可以在不使用ARC下,可用.

__weak:LLVM 编译器 3.0版本之后,可以在不使用ARC下,可用.

__strong:LLVM 编译器 3.0版本之后,可以在不使用ARC下,可用.

__block:块变量。

等.

那Clang编译器支持的所有宏定义可通过

clang -dM -E - < /dev/null
