以前在像网易上写博客，现在开始在**github&gitcafe**上写博客:

**octopress+github|octopress+gitcafe**

本博客对网易博客并没有做迁移，主要是针对以前项目及学习中做一些梳理总结，更多的是个人笔记而已。

* 博客编辑主要使用[mou](http://mouapp.com)。
* 博客内容**技术方面**为主,偶有一些其它方面的点滴记录。
